# Flask 專案教學計畫

## 專案總覽

本計畫旨在引導無程式基礎的學生，透過拆解一個已完成的 Flask Web 應用程式，循序漸進地學習 Python、HTML、CSS 和 JavaScript 的核心技能，並最終學會使用 Chart.js 進行數據視覺化。

- **專案核心**：一個包含資料視覺化（KNN、迴歸）的互動式網站。
- **教學對象**：程式設計初學者。
- **教學目標**：完成專案的複製，並理解其背後的技術原理。
